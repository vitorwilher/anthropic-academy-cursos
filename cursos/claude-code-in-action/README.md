# Claude Code in Action — Anthropic Academy

**Status:** ✅ Concluído

Curso prático e operacional do Claude Code: o sequel aprofundado do Claude Code 101.
Cobre fluxos de trabalho do dia a dia — gerenciamento de contexto (`/init`, `CLAUDE.md`,
`@` menções), Plan Mode e Thinking Mode, atalhos e comandos, **comandos customizados**,
**servidores MCP**, **integração com GitHub**, **hooks** e o **Claude Code SDK** (uso
headless/CI). Exemplos com sabor de economia/finanças (pipelines de dados).

Fonte: <https://anthropic.skilljar.com/claude-code-in-action>
