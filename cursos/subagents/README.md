# Introduction to Subagents — Anthropic Academy

**Status:** ✅ Concluído

Curso sobre **subagents** no Claude Code: assistentes especializados que rodam em uma
**janela de contexto separada**, recebem uma tarefa, trabalham de forma independente e
devolvem só o resumo. Cobre o que é um subagent, como criá-lo (arquivo markdown com
frontmatter `name`/`description`/`tools`/`model` + system prompt, ou via comando
`/agents`), como projetar subagents eficazes e como usá-los bem — delegação automática,
`@-mention`, paralelismo e quando **não** usar.

Fonte da grade: página oficial do curso na Anthropic Academy
(https://anthropic.skilljar.com/introduction-to-subagents) + documentação oficial
(https://code.claude.com/docs/en/sub-agents).
