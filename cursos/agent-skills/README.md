# Introduction to Agent Skills — Anthropic Academy

**Status:** ✅ Concluído

Curso sobre **Agent Skills**: instruções reutilizáveis empacotadas em um arquivo
`SKILL.md` (com frontmatter YAML), que o Claude aciona automaticamente na hora certa.
Cobre o que é uma skill, como criar a primeira, configuração e skills com múltiplos
arquivos (progressive disclosure, scripts, `allowed-tools`), como skills se comparam a
outros recursos do Claude Code, e como compartilhar/distribuir skills num time.

## Grade do curso

- [ ] 1. What are skills?
- [ ] 2. Creating your first skill
- [ ] 3. Configuration and multi-file skills
- [ ] 4. Skills vs. other Claude Code features
- [ ] 5. Sharing skills
- [ ] 6. Troubleshooting skills
